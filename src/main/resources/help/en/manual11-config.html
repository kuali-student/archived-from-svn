Configuration

You can change some settings at the FAR > Settings menu entry. With the present version you may:
  • Change the default character set and add new character encodings. To add a new encoding simply type the encoding name (or alias) into the text field of the drop down list. See http://www.iana.org/assignments/character-sets for a list of official encodings. Note however that not all those encodings may be available on your machine. FAR will tell you.
  • Select the "group reference indicator" that is used for replacement operations with regular expressions.
  • Prevent backups. Usually you do not want this. If you unset this flag, you can not undo any replacement operation any more. It may be useful in rare cases where you have limited space but a lot of (large) files to change. You will be prompted before any replacement operation if you unselect this.
  • Set the directory where backup files are to be stored. This has the same effect as the "Backup Directory" field on the "Replace" tab.
  • Select the information that should be displayed in the last column of the result table. You may choose between warnings on read-only files, file size and the last modified date, or to display nothing at all.
Any changes to the configuration are immediately effective. Note that the "prevent backup" flag can only be unset for the duration of a single session (an invocation of the application). All other configuration values are of course stored on disk when you close the application.

The core API. If you are looking for the main class, go to {@link net.pandoragames.far.ui.swing.FindAndReplace FindAndReplace} in the Swing package.

To use FAR from within another application you might want using the classes in this package directly. In this case {@link net.pandoragames.far.FileSelector FileSelector} and {@link net.pandoragames.far.FileMatcher FileMatcher} should be your starting points.
Considder using {@link net.pandoragames.far.ui.UIBean UIBean} or {@link net.pandoragames.far.ui.UIFace UIFace} though.
